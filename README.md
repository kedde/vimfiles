# vimfiles

My portable vim configuration

## Install

Linux/Mac:

```
git clone https://github.com/lyphtec/vimfiles.git ~/.vimfiles
cd ~/.vimfiles
./install.sh
```

Windows (from Powershell prompt):

```
cd ~
git clone https://github.com/lyphtec/vimfiles.git
cd vimfiles
.\install.bat
```

## Uninstall

Linux/Mac:

```
cd ~/.vimfiles
./uninstall.sh
```

Windows (from Powershell prompt):

```
cd ~
.\vimfiles\uninstall.bat
```
